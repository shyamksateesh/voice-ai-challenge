# LyrAssist - GitHub Pages Demo

This folder contains the static demo version of LyrAssist for GitHub Pages deployment.

## What's Included

- **index.html** - Static demo interface
- **demo-assets/** - Pre-processed demo video and assets

## Limitations

This demo version is **static-only** and cannot:
- Process new audio/video files
- Run AI models (Whisper, Demucs, WhisperX)
- Upload files to a backend
- Generate new lyric videos

## What It Shows

- Full UI/UX design
- Interactive lyrics feature (click on words to jump to timestamps)
- Pre-processed example video ("Almost Lover" by A Fine Frenzy)
- All frontend functionality

## For Full Functionality

To process your own files with AI transcription and video generation:
1. Visit the main repository: https://github.com/shyamksateesh/LyrAssist
2. Follow the installation instructions
3. Run locally on your machine

This allows you to:
- Upload any audio/video file
- Process with Whisper AI
- Optional vocal separation
- Generate custom lyric videos
- Download results
